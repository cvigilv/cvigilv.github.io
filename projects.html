Projects

I believe that the best way to learn is by doing, and that the best way to do is by sharing. With that in mind, here are some of the projects I'm currently working on:

Science (my publications):

  • De novo prediction of drug targets and candidates by chemical similarity-guided network-based inference
  • ResidueFisher, a bioinformatics protocol that aims at mining information at the sequence and structure level of the protein chain to detect possible evolutionarily conserved residues

Neovim:

  • diferente.nvim, a plugin for enhancing the UX of Git commits in Neovim, inspired in commitia.vim.
  • esqueleto.nvim, a plugin for managing template files in Neovim.
  • denote.nvim, a plugin for simple notes for Neovim with an efficient file-naming scheme.

Julia language:

  • SimSpread.jl, a software package implements the SimSpread algorithm for link prediction in graphs.
  • HyperdimensionalComputing.jl, a software package for hyperdimensional computing (HDC) in Julia.
  • KEGGAPI.jl, a library for working with the KEGG API.
  • HyperGen.jl, a reimplementation of HyperGen in Julia.

Miscellaneous:

  • recetario, a compendium of miscellaneous script for all things science, ML, DS, Julia, etc.
